Published Games

This section shows my published games hosted by Itch.io. Those items are fetched using my NodeJS backend that consumes the Itch.io API and filters the games to get only the published ones, as the itch.io returns all hosted games, even the ones on draft mode.
The items might take a little to load as my backend is hosted by heroku with a free license that after sometime without using it enters on a hibernate state and it awakes when it's requested again.
All of my games are developed using the Unity game engine and the C# language.
